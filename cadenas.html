Preguntas de Interés

¿Qué es una cadena en JS? Una cadena de JavaScript es cero o más caracteres escritos entre comillas, ejemplo: let text "Hola Mundo";

¿Que utlilizamos para crear cadenas? Se pueden utilizar comillas simples o dobles, tambien puede usar comillas dentro de una cadena, siempre que no coincidan con las comillas que rodean la cadena
